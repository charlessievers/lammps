dynamical_matrix command :h3

[Syntax:]

dynamical_matrix group-ID style args keyword value ... :pre

group-ID = ID of group of atoms to use in dynamical matrix calculation :ulb,l
style = {regular} or {eskm} :l
  {regular} args = gamma 
    gamma = finite difference displacement length (distance units)
  {eskm} args = gamma 
    gamma = finite difference displacement length (distance units)

zero or more keyword/value pairs may be appended :l
keyword = {file} or {binary} :l
  {file} value = output_file
    output_file = name of file to dump the dynamical matrix into
  {binary} values = {yes} or {no} or {gzip}:pre
:ule

[Examples:]

dynamical_matrix 1 regular 0.000001
dynamical_matrix 1 eskm 0.000001
dynamical_matrix 3 regular 0.00004 file dynmat.dat
dynamical_matrix 5 eskm 0.00000001 file dynamical.dat binary yes :pre

[Description:]

Calculate the dynamical matrix of the selected group using finite difference.

[Restrictions:]

None

[Related commands:]

"fix phonon"_fix_phonon.html

[Default:]

The option defaults are file = "dynmat.dyn", binary = no.
