third_order command :h3

[Syntax:]

third_order group-ID style args keyword value ... :pre

group-ID = ID of group of atoms to use in third order tensor calculation :ulb,l
style = {regular} or {ballistico} :l
  {regular} args = gamma
    gamma = finite difference displacement length (distance units)
  {ballistico} args = gamma
    gamma = finite difference displacement length (distance units)

zero or more keyword/value pairs may be appended :l
keyword = {file} or {binary} :l
  {file} value = output_file
    output_file = name of file to dump the third order tensor into
  {binary} values = {yes} or {no} or {gzip} :pre
:ule

[Examples:]

dynamical_matrix 1 regular 0.000001
dynamical_matrix 1 ballistico 0.000001
dynamical_matrix 3 regular 0.00004 file dynmat.dat
dynamical_matrix 5 ballistico 0.00000001 file third_order binary yes :pre

[Description:]

Calculate the third order tensor of the selected group using finite difference.

[Restrictions:]

None

[Related commands:]

"dynamical matrix"_dynamical_matrix.html

[Default:]

The option defaults are file = "third_order.dat", binary = no.
